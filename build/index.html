Software Libre Taldea

hasiera · ekitaldiak · faq

Software Libre Taldea, 2021-2022 kurtsoan sortutako talde bat da, Donostiako Informatika Fakultateko lagun batzuen ekimenez.

Software, hardware eta kultura librean dugun interesak bultzatu gintuen talde hau sortzera, gai hauen inguruan eztabaidatzeko talde baten beharra ikusten genuen.

Nahiz eta taldearen sorrera Informatika Fakultatean eman zen, taldearen helburuak betetzeko, fakultatetik kanpo eta fakultatearekin lotuta ez dagoen jendearekin jardun nahi dugu.

Gure Helburuak

  • Software librearen erabilera bultatzea eta aldarrikatzea fakultate barnean eta kanpoan.
  • Software librearen garrantzia eta software propietarioaren kaltea ikusaraztea.
  • Software librea eta bestelako teknologia libreen inguruan eztabaidatzeko talde inklusiboa izatea.
  • Software eta kultura librearen inguruko ekitaldiak, tailerrak edo bestelako ekintzak antolatzea.

Zer da Software Librea?

«Software librea» erabiltzaileen eta komunitatearen askatasuna errespetatzen duen softwarea da. Labur esanda, erabiltzaileek softwarea exekutatu, kopiatu, banatu, aztertu, eraldatu eta hobetzeko askatasuna bermatzen du. Alegia, software librea askatasunari lotuta dago, ez prezioari. Inglesez esanda, «free softwareko» free hitzak “askatasuna” hitzaren adiera hartzen du (freedom) ez “doakoa” hitzarena (free beer). Horregatik, batzuetan, ingelesez «libre software» erabiltzen da, ez pentsatzeko doako softwarea esan nahi dutela.

Posible da dirua ordaindu izana programa libre bat eskuratzeko, edo ez. Baina programa hori nola lortu zenuen kontuan hartu gabe, beti izango duzu softwarea eraldatzeko askatasuna, baita saltzekoa ere.
